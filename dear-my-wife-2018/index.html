2017/12/01

諸君、私は嫁ちゃんが好きだ 2017

これは 妻・夫を #愛してる ITエンジニア Advent Calendar 2017 1日目の記事です。
発起人にして残り3時間を切るタイミングでの公開……!!

はじめに

昨年は その1, その2 とPart2まで生まれた本アドベントカレンダーですが、今年2017年は初日にしてようやく半分が埋まった程度です。

既婚者クラスタのエンジニア諸氏におかれましては、年末のこのタイミングでいくら惚気けても誰にも怒られないので、えいやで書き連ねてみると良いのではないでしょうか。

本アドベントカレンダーにルールはありません。本当に575で詠う形でも構いません。もう少し書けるなら57577で14文字多く詠ってください。そして1ツイート程度から少しずつ長くしていく形でも構いません。
妻・夫に対して面と向かって「愛している」とテキストとして綴らないまでも、ちょっとした感謝の気持ちとか、普段なかなか言えないようなこととか、なんというかそんなポカポカするような何かがクリスマスを迎える日まで続けばいいなーと思いながら始めます。

今年やってきた2つの転機

Baby was Born 👶

妻は「母」になり、私は「父」になりました！
大好きな嫁ちゃんを「妻」として愛し、「母」として尊敬しています！

子育て的なアレコレは 子育てエンジニア・クリエイター (これも埋まっていない……!!) に書き連ねる予定ですが、赤さんが生まれて「日常」は新たなステージに突入しました。まさに『「子供を持った時点で自分は脇役になる」って話』の通りだなーという気持ちになりつつ、だからといって決してネガティブには捉えていなくて、より嫁ちゃんのことを好きになった気がしています:D

とりあえず Octocat One-Piece が最高に可愛すぎるので、興味のある方は今すぐ注文するといいと思います！

なお、嫁ちゃんも同じ界隈の属性を持っている人なので、普段からお揃いの GitHub Tシャツ や、Maker Faire Tokyo 2017 のTシャツを着ていたりします。オソロ！オソロ！

転職 📱

嫁ちゃん同様に愛してやまない赤さんがこの世に放たれた翌月末、私は当日勤めていた企業を退職し、有給休暇を消化することもなく、現職で働き始めました！

転職的なアレコレは 退職者 アドベントカレンダー (こっちは埋まっている……!!) に書き連ねる予定ですが、がっつり育休まではいかなくとも、結構大胆に残してしまった有給休暇を無理にでも使うべきだった、と少し後悔と申し訳なさを持ちつつ、そんな状況でも「やりたいことをやるために」転職を応援してくれた嫁ちゃんには本当に感謝しています。

諸君、私は嫁ちゃんが好きだ

そんなわけで昨年に引き続き「諸君、私は嫁ちゃんが好きだ」と高らかに宣言して〆たいと思います。おしまい。

明日 2 日目は Hirotaka Nakajima さんです！
